# Chemical Plant Energy Optimization App
A Flask-based web app to optimize energy use in chemical processes (e.g., boilers).

## Features
- Input process parameters.
- Calculate energy consumption and savings.
- Visualize results with Plotly charts.
- Modern Bootstrap UI with chemical engineering color scheme.

## Installation
1. Clone the repo: `git clone <repo-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`

## Usage
1. Enter boiler parameters (feedwater temperature, steam pressure, fuel flow, efficiency).
2. View optimized energy consumption and savings.
3. Explore interactive charts.

## Tech Stack
- Python/Flask
- Bootstrap 5
- Plotly
- CSS with variables
